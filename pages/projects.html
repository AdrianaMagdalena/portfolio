PixartAdri

Artist's showcase website

Features:

The website was created to serve as an artist's showcase.

The homepage contains an introduction hero section, a short preview of gallery content and a quote section.

The navigation pane contains links to different sections of hompage, a link to separate gallery and social media links. Currently only Instagram link guides outside the website, the rest of SM links were added as placeholders.

The gallery sub-page contains all of artist's art. Each piece can be opened in and overlay modal in preview carousel.

  • Reworking existing projects into mobile-first design
  • Creating picture prewiev carousel in JavaScript
